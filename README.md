# Resumo Clínico Inteligente - Clinic

Landing page em HTML/CSS para divulgação do recurso **Resumo Clínico Inteligente**.

## Como publicar no GitHub Pages

1. Crie um repositório no GitHub.
2. Envie o arquivo `index.html` e a pasta `assets`.
3. Acesse **Settings > Pages**.
4. Em **Branch**, selecione `main` e `/root`.
5. Clique em **Save**.

Depois de alguns minutos, o GitHub Pages irá gerar o link público.
